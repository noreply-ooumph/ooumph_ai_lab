Complete Curriculum

8 comprehensive modules spanning 52 weeks with stage-gates and assignments

M1

AI Foundations & Python

Weeks 1-4 | 56 hours

Python environments, fundamentals, NumPy/pandas, Jupyter, Git workflows, unit testing

Gate G1: Python Quiz ≥70%
M2

Data Engineering & Annotation

Weeks 5-8 | 56 hours

Data scraping, cleaning, CVAT/Label Studio, inter-annotator agreement, dataset splits

Gate G2: Dataset IAA ≥0.7
M3

Classical ML & Evaluation

Weeks 9-12 | 56 hours

Regression, random forests, XGBoost/LightGBM, metrics (F1/ROC/AUC), Fairlearn bias audits

Gate G3: Model Card v0
M4

Deep Learning (CV Track)

Weeks 13-18 | 84 hours

PyTorch, CNN (ResNet/MobileNet), YOLOv8, U-Net, OCR (PaddleOCR), quantization, TensorRT

Gate G4: Quantised CV Model
M5

NLP & Regional-Language AI

Weeks 19-22 | 56 hours

Tokenization (BPE), Odia/Hindi specifics, Transformer architecture, IndicBERT, IndicTrans2

Gate G5: Odia NLP F1 ≥0.7
M6

Generative AI, RAG & Agents

Weeks 23-28 | 84 hours

LLM APIs (Claude, Llama), prompt engineering, RAG (LangChain/LlamaIndex), LangGraph, LoRA

Gate G6: 3-intent Agent + HITL
M7

Edge AI, IoT & MLOps

Weeks 29-36 | 112 hours

Jetson/Coral, IoT (MQTT), Docker/K8s, CI/CD (GitHub Actions, MLflow), Prometheus/Grafana

Gate G7: K8s + Monitoring Live
M8

DAO Capstone + Deployment

Weeks 37-52 | 224 hours

PRD, baseline, agent wrapper, frontend MVP, customer pilot, pricing model, Demo Day

Gate G8: Pilot LoI + Joint Panel ✨
